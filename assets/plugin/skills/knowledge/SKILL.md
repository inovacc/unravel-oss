---
name: knowledge
description: Generate structured knowledge source from any application
---

Use `unravel_knowledge` to generate a comprehensive knowledge directory from any analyzed application. The output includes:
- Communication patterns (APIs, protocols, endpoints)
- Security configuration (CSP, signing, permissions)
- Dependencies (libraries, frameworks, SDKs)
- Telemetry (analytics, attribution, crash reporting)

Use `unravel_knowledge_diff` to compare two knowledge directories and detect behavioral changes between app versions.
